    Quiz-its Delish

Overview

Quiz-its is a responsive CRUD product that users can login, select a quiz to play, get results, save and delete results.

The project was complete within a two-week deadline. The requirements include developing user personas, conducting user interviews and research for insights to develop the product, creating wireframes, prototypes, using Firebase for database storage and user authentication, and GitHub project tracking. Other tools and techologies used to build the application: HTML, CSS, SASS, jQuery, JavaScript, Bootstrap, Browserify, Grunt, GitHub, Extensio, Adobe XD,Photoshop,and Illustrator.

Challenge Statement

How might we create a quick and fun activity for users to engage with to take a break from everyday stress.

User Research Interviews

Interviewed 7 users of various ages ranging from 20 something millenials to 50-year-old professionals that enjoy playing social media quizzes or participating in trivia game.

Research Insights

  • Users like to test their general knowledge skills.
  • Users like to keep the results to themselves and seldom share their quiz results on social media.
  • User reasons for taking a quiz or playing trivia vary from liking learning new fun facts to learning more about themselves through answering the questions.
  • Users spend about 10 minutes a day/week engaged in a social or trivia quiz.

Project Reflections

  • Creating quizzes from scratch take a lot of time. I should have used more of the trivia API content to create more quizzes.
