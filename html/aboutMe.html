PHONETHIP LIU HOBSON

Front-end web developer & UX/UI designer + Graphic designer & Illustrator

Former award-winning newspaper designer and visual journalist with two decades of print experience taking the leap to online design and developement. I have always been interested in creating a visually thoughtful user experience in print but now I want to architect websites that are functional, easy to navigate and beautifully designed.

My journey began as an apprentice at Nashville Software School where I immersived myself in a six-month long, full-time software development bootcamp focusing on front-end development, user experience, and user interaction design fundamentals.

Development experience involves daily hands on application of development fundamentals and principles through group and individual projects reflecting real world business problems using source code version control with GitHub as well as utilizing Github Projects & Issue Tracking for project management/tracking.

Implement JavaScript fundamentals leveraging DRY, modular, readable code and reusable components. Built single-page applications in the React framework leveraging HTML, CSS, and JavaScript. Utilize task automation with Grunt: Linting with JSHint, module bundling with Browserify, SASS compilation. Styled applications with the CSS framework Bootstrap and wrote custom styles with SASS, CSS Grid and Flexbox. Deployed applications through Firebase and GitHub Pages.

UX and UI design involves creating user stories based on assessment of user experience and workflow to produce visual & interactive design with strategic use of typography, layout, color, motion, and behavior. Convey interactive story ideas and complex interactions with wireframes or prototypes. Applied corporate style guides and pattern libraries to new applications and web sites and focused on user experience with intentional design around speed, efficiency, and smooth functionality.
